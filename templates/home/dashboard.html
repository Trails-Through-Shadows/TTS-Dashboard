Trails Through Shadows

Trails Through Shadows is a narrative tabletop game with hybrid parts - you need both our game and your phone/laptop/projector. It's heavily inspired by Gloomhaven and Forgotten Waters, which can be seen in the combat system and branching paths based on the player's choices in the narrative.

The game is played in two parts - the physical board game and the digital companion app. The board game is where the players will move their characters, fight monsters, and make decisions that will affect the story. The companion app will provide the narrative, the branching paths, and the combat system.

Project Sections / Links

play.tts-game.fun is an online game visualizer that will show the game board and the narrative playground for player journey. You need to use this during the game to see what's happening in the game using your license key on any kind of device.
api.tts-game.fun is the API providing all the game logic, data and the narrative for the game. You can use this API to create your own game visualizer or play the game in any other way you want.
docs.tts-game.fun is the documentation for the whole project. You can see there all the game rules, how to play the game all the game data and the API documentation.
